## Introduction

I kept getting fed up with different "ai" cli tools, so I just built one.

```console
$ cargo install chat

$ OPENAI_API_KEY=sk... chat "Write an owl Hiku"
Silent wings take flight,
Eyes of wisdom in the night,
Owl watches, unseen.
```
